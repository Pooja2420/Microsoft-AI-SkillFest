The Kusto Python Client library lets you query your database using Python.
The library is Python 2.x/3.x compatible. It supports all data types using the Python DB API interface.
You can use the library, for example, fromJupyter Notebooksthat are attached to Spark clusters,
including, but not exclusively,Azure Databricksinstances.
Kusto Python Ingest Clientis a python library that lets you send, or ingest, data to your database.
