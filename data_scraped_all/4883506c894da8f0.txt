You can get the file hash for your download by running the following PowerShell cmdlet.
Get-FileHash -Path 'W:\temp\AzureDevOps2022.iso' -Algorithm SHA256
Get-FileHash -Path 'W:\temp\AzureDevOps2022.iso' -Algorithm SHA256
You can then compare the value that you get to the expected hash provided on this page to verify its integrity. If the hash doesn't match, your download may be corrupted and you should download the file again.
Note
The hashes provided on this page only apply to downloads obtained fromAzure DevOps Server Downloads.
