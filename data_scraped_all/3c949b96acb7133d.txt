Language support for Language Detection
Article
2024-11-21
4 contributors
In this article
Use this article to learn which natural languages that language detection supports.
The Language Detection feature can detect a wide range of languages, variants, dialects, and some regional/cultural languages, and return detected languages with their name and code. The returned language code parameters conform toBCP-47standard with most of them conforming toISO-639-1identifiers.
If you have content expressed in a less frequently used language, you can try Language Detection to see if it returns a code. The response for languages that can't be detected isunknown.
