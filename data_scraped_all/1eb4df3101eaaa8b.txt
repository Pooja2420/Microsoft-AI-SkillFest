Edit the Hide snapshot path option of Azure NetApp Files
Article
2025-01-29
3 contributors
In this article
The Hide snapshot path option controls whether the snapshot path of a volume is visible. During the creation of anNFSorSMBvolume, you can specify whether the snapshot path should be hidden. After creating the volume, you can edit the Hide snapshot path option as needed.
Note
For adestination volumein cross-region replication, the Hide snapshot path option is enabled by default. The setting isn't modifiable.
Steps
To view the Hide snapshot path option setting of a volume, select the volume. TheHide snapshot pathfield shows whether the option is enabled.
To edit the Hide Snapshot Path option, selectEditon the volume page. Modify theHide snapshot pathoption as needed.
