Azure Private Link availability
Article
2025-03-25
34 contributors
In this article
Azure Private Link enables you to access Azure PaaS Services (for example, Azure Storage and SQL Database) and Azure hosted customer-owned/partner services over aprivate endpointin your virtual network.
Important
Azure Private Link is now generally available. Both Private Endpoint and Private Link service (service behind standard load balancer) are generally available. For known limitations, seePrivate EndpointandPrivate Link Service.
