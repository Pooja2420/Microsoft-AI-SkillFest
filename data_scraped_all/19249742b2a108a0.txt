Supported logs for Microsoft.Dashboard/grafana
Article
2025-04-16
2 contributors
In this article
The following table lists the types of logs available for the Microsoft.Dashboard/grafana resource type.
For a list of supported metrics, seeSupported metrics - Microsoft.Dashboard/grafana
GrafanaLoginEvents
Login events for an instance of Azure Managed Workspace for Grafana including user identity, user Grafana role (in success) and detailed message (in failure).
GrafanaUsageInsightsEvents
Usage insights events for an instance of Azure Managed Workspace for Grafana.
