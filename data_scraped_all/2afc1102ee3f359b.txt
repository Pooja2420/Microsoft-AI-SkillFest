SQL Insights,retiredon 31 December 2024, is no longer supported.
For Azure SQL Database, and Azure SQL Managed Instance,database watcheris the recommended monitoring solution for scenarios that require low data collection latency, estate-level monitoring, comprehensive monitoring data including query-level details, and support for advanced analytics on collected monitoring data. At this time, database watcher supports Azure SQL Database and Azure SQL Managed Instance. For other monitoring options, reviewMonitor and tune Azure SQL.
